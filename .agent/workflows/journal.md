# /journal - Viết Nhật Ký

Viết journal entries cho dự án.

1. Sử dụng `journal-writer` subagent
2. Explore memories và recent code changes
3. Write journal entries

## Format

- Concise và focused
- Key events, changes, impacts, decisions
- Lưu tại: `./docs/journals/`
